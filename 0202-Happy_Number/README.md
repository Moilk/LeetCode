## 题目描述

编写一个算法来判断一个数是不是“快乐数”。

一个“快乐数”定义为：对于一个正整数，每一次将该数替换为它每个位置上的数字的平方和，然后重复这个过程直到这个数变为 1，也可能是无限循环但始终变不到 1。如果可以变为 1，那么这个数就是快乐数。

**示例: **

&emsp;**输入:** `19`  
&emsp;**输出:** `true`  
&emsp;**解释:**  
&emsp;`12 + 92 = 82`  
&emsp;`82 + 22 = 68`  
&emsp;`62 + 82 = 100`  
&emsp;`12 + 02 + 02 = 1`
