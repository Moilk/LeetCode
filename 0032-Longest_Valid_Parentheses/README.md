## 题目描述

给定一个只包含 `(` 和 `)` 的字符串，找出最长的包含有效括号的子串的长度。


**示例 1:**

&emsp;**输入:** "(()"  
&emsp;**输出:** 2  
&emsp;**解释:** 最长有效括号子串为 "()"  

**示例 2:**

&emsp;**输入:** ")()())"  
&emsp;**输出:** 4  
&emsp;**解释:** 最长有效括号子串为 "()()"
