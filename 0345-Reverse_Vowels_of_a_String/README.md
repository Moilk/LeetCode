# 345. 反转字符串中的元音字母

## 题目描述

编写一个函数，以字符串作为输入，反转该字符串中的元音字母。

**示例 1:**

&emsp;**输入:** `"hello"`  
&emsp;**输出:** `"holle"`

**示例 2:**

&emsp;**输入:** `"leetcode"`  
&emsp;**输出:** `"leotcede"`

**说明:**

元音字母不包含字母"y"。
