# 14. 最长公共前缀

编写一个函数来查找字符串数组中的最长公共前缀。

如果不存在公共前缀，返回空字符串 ""。

**示例 1:**

&emsp;**输入:** `["flower","flow","flight"]`  
&emsp;**输出:** `"fl"`  

**示例 2:**

&emsp;**输入:** `["dog","racecar","car"]`  
&emsp;**输出:** `""`  
&emsp;**解释:** 输入不存在公共前缀。  
&emsp;**说明:**  

所有输入只包含小写字母 `a-z` 。
