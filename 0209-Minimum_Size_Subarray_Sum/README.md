## 题目描述

给定一个含有 n 个正整数的数组和一个正整数 s ，找出该数组中满足其和 ≥ s 的长度最小的连续子数组。如果不存在符合条件的连续子数组，返回 0。

**示例: **

&emsp;**输入:** `s = 7, nums = [2,3,1,2,4,3]`  
&emsp;**输出:** 2  
&emsp;**解释:** 子数组 `[4,3]` 是该条件下的长度最小的连续子数组。

**进阶:**

如果你已经完成了 `O(n)` 时间复杂度的解法, 请尝试 `O(n log n)` 时间复杂度的解法。
