## 题目说明

给定正整数 *n*，找到若干个完全平方数（比如 `1`, `4`, `9`, `16`, ...）使得它们的和等于 *n*。你需要让组成和的完全平方数的个数最少。

**示例 1:**

&emsp;**输入:** `n = 12`  
&emsp;**输出:** `3`  
&emsp;**解释:** `12 = 4 + 4 + 4`.  

**示例 2:**

&emsp;**输入:** `n = 13`  
&emsp;**输出:** `2`  
&emsp;**解释:** `13 = 4 + 9`.
