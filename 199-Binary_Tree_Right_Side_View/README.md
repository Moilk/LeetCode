## 题目描述

给定一棵二叉树，想象自己站在它的右侧，按照从顶部到底部的顺序，返回从右侧所能看到的节点值。

**示例:**

&emsp;**输入:** `[1,2,3,null,5,null,4]`
&emsp;**输出:** `[1, 3, 4]`
&emsp;**解释:**

```
   1            <---
 /   \
2     3         <---
 \     \
  5     4       <---
```
