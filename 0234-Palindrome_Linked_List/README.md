## 题目描述

请判断一个链表是否为回文链表。

**示例 1:**

&emsp;**输入:** `1->2`  
&emsp;**输出:** `false`

**示例 2:**

&emsp;**输入:** `1->2->2->1`  
&emsp;**输出:** `true`

**进阶：**

你能否用 O(n) 时间复杂度和 O(1) 空间复杂度解决此题？
