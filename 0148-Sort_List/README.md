# 148. 排序链表

## 题目描述

在 `O(n log n)` 时间复杂度和常数级空间复杂度下，对链表进行排序。

**示例 1:**

&emsp;**输入:** `4->2->1->3`  
&emsp;**输出:** `1->2->3->4`

**示例 2:**

&emsp;**输入:** `-1->5->3->4->0`  
&emsp;**输出:** `-1->0->3->4->5`
